# 📋 Contact Book CLI

A beginner Python project — a command-line app to manage contacts stored in a dictionary.

---

## 🚀 How to Run

```bash
python contact_book.py
```

---

## 📌 Features

- Add a new contact (name, phone, email)
- Search for a contact by name
- Update an existing contact's phone, email, or both
- Delete a contact with confirmation
- View all saved contacts

---

## 🖥️ Demo

```

5. View all contacts
```
